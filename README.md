## About

A module which posts very dank memes. What else?

[![NPM](https://nodei.co/npm/memie-generator.png)](https://nodei.co/npm/memie-generator/)

## How to setup 101

``npm install memie-generator``

```
const memes = require('memie-generator');

memes.randomImages();
memes.allImages();

memes.randomWords();
memes.allWords();
```
